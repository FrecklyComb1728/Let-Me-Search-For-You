# Let Me Search That For You

一个有趣的搜索引导工具，解决手残党不会使用浏览器的问题（）。支持百度、Google和Bing三大搜索引擎。

## 功能特点

- 支持三大主流搜索引擎：
  - 百度 (Baidu)
  - 谷歌 (Google)
  - 必应 (Bing)
- 简洁直观的用户界面
- 生成可分享的搜索链接
- 支持一键复制搜索链接
- 支持搜索预览功能

## 使用方法

1. 选择你想使用的搜索引擎版本（百度/Google/Bing）
2. 在搜索框中输入你的问题
3. 点击搜索按钮
4. 获取生成的链接，可以选择：
   - 复制链接分享给他人
   - 点击预览查看搜索效果

## 技术栈

- HTML5
- CSS3
- jQuery
- Clipboard.js
